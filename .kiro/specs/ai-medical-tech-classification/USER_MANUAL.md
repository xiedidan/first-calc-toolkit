# 医技智能分类分级 - 用户手册

## 1. AI接口配置

### 配置步骤
1. 进入"系统设置" → "AI接口管理"
2. 填写配置信息：
   - **API端点**：DeepSeek API地址（如 `https://api.deepseek.com/v1`）
   - **API密钥**：从DeepSeek获取的密钥
   - **提示词模板**：使用占位符 `{item_name}` 和 `{dimensions}`
3. 点击"测试配置"验证连接
4. 保存配置

### 提示词示例
```
请根据以下医技项目名称，从给定的维度列表中选择最合适的分类：

项目名称：{item_name}

可选维度：
{dimensions}

请返回JSON格式：{"dimension_id": 维度ID, "confidence": 确信度(0-1)}
```

## 2. 创建分类任务

### 操作步骤
1. 进入"智能分类分级" → "分类任务管理"
2. 点击"创建分类任务"
3. 配置任务：
   - **任务名称**：便于识别的名称
   - **收费类别**：选择需要分类的类别（可多选）
   - **模型版本**：选择目标模型版本
4. 提交后系统自动开始AI分类

### 进度监控
- 任务列表显示实时进度
- 处理中断可点击"继续处理"
- 完成后显示"查看预案"按钮

## 3. 预案调整

### 查看预案
1. 任务完成后点击"查看预案"
2. 预案显示所有项目的AI分类结果

### 调整维度
1. 筛选低确信度项目（如 <0.5）
2. 点击"调整"按钮修改维度
3. 调整后项目标记为"已调整"
4. 保存预案并输入预案名称

### 排序和筛选
- 按确信度排序：升序/降序
- 筛选范围：设置确信度阈值

## 4. 提交流程

### 提交预览
1. 点击"提交预案"
2. 系统分析影响范围：
   - **新增**：项目在目标维度中不存在
   - **覆盖**：项目已存在，将更新维度归属
3. 确认统计数字

### 正式提交
1. 确认预览无误后点击"确认提交"
2. 系统批量更新维度目录
3. 提交成功后预案状态变为"已提交"

## 5. 常见问题

### Q: API调用失败怎么办？
A: 检查API端点和密钥是否正确，使用"测试配置"功能验证。

### Q: 任务中断如何继续？
A: 点击任务列表中的"继续处理"按钮，系统从断点继续。

### Q: 如何提高分类准确性？
A: 优化提示词模板，增加更多上下文信息和示例。

### Q: 预案可以修改吗？
A: 草稿状态的预案可以修改，已提交的预案不可修改。

### Q: 覆盖操作会影响什么？
A: 覆盖会更新项目的维度归属，原有分类将被替换。
