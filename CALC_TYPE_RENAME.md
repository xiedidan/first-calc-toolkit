# 算法类型命名统一 - 修改说明

## 📋 修改内容

### 字段名称修改
- **"计算类型"** → **"算法类型"**

### 取值修改
- **"统计型"** (statistical) → **"指标"**
- **"计算型"** (calculational) → **"目录"**
