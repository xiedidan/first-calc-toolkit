## 📋 完整测试流程

### 前置条件
- ✅ 已安装Python 3.12
- ✅ 已安装依赖：`pip install -r backend/requirements.txt`
- ✅ 已有计算结果数据

### 步骤1：启动后端
```bash
cd backend
uvicorn app.main:app --reload
```

### 步骤2：启动前端
```bash
cd frontend
npm run dev
```

### 步骤3：测试导出
1. 访问：http://localhost:3000
2. 登录系统
3. 进入"评估结果"页面
4. 选择评估月份
5. 点击"导出汇总表"按钮
6. 自动下载Excel文件

## ❓ 常见问题

### Q: 测试脚本运行失败？
A: 确保已安装openpyxl：`pip install openpyxl`

### Q: 前端下载失败？
A: 检查后端是否正常运行，查看浏览器控制台错误

### Q: Excel文件打不开？
A: 确保使用Excel 2016+或WPS Office
