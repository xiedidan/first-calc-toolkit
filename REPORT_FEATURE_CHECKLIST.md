## 🔧 启动步骤

1. **运行数据库迁移**
   ```bash
   cd backend
   alembic upgrade head
   ```

2. **启动后端服务**
   ```bash
   cd backend
   uvicorn app.main:app --reload
   ```

3. **启动Celery Worker**
   ```bash
   cd backend
   celery -A app.celery_app worker --loglevel=info --pool=solo
   ```

4. **启动前端服务**
   ```bash
   cd frontend
   npm run dev
   ```
